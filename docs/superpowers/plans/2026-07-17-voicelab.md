# VoiceLab Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A one-click self-deploy Netlify app (neo-styled) where each attendee runs their own private tool with two cards — Testing (Retell webhook → rate calls) and Transcription (upload audio → diarized transcript → saved library) — organized by Projects.

**Architecture:** Static React+Vite SPA served by Netlify, with Netlify Functions (v2, ESM) for all server work and Netlify Blobs for storage (zero DB setup). Data is namespaced by project. The Testing webhook receiver ports FilmBros' `retell-call-event` parse/merge logic; Transcription uploads browser-direct to Deepgram (bring-your-own key) via a short-lived grant token so large audio never hits the ~6 MB function-body limit.

**Tech Stack:** React 18, Vite, TypeScript, Tailwind CSS, `@netlify/blobs`, Netlify Functions v2, `lucide-react`, `framer-motion`, `sonner`, Deepgram (transcription), Vitest (unit tests for pure logic).

## Global Constraints

- **Single-tenant, self-deployed.** No user accounts/roles/invites. Each deploy is one owner's private instance.
- **Zero required setup at deploy.** All env vars are OPTIONAL: `DEEPGRAM_API_KEY` (enables Transcription), `SITE_PASSWORD` (gates the app), `RETELL_API_KEY` (enables optional webhook HMAC). App must run and be useful with none set (Transcription shows a "add your key" state).
- **No secrets in the repo.** The public deploy repo contains zero credentials.
- **Netlify Blobs is the only datastore.** No Postgres/Supabase.
- **Storage keys:** store name `voicelab`; keys `projects`, `calls/<projectId>/<callId>`, `transcripts/<projectId>/<transcriptId>`.
- **Rating scale (exact ids):** quality `bad | weak | good | perfect`; latency `fast | normal | slow`; note required only for `bad`/`weak`.
- **Netlify Functions are v2 ESM:** each file `export default async (req, context) => Response`. Read query via `new URL(req.url).searchParams`; raw body via `await req.text()`.
- **Neo design is lifted verbatim from FilmBros** `repo/Code/dashboard` — no new hues, no new visual language.
- **Frontend (UI) tasks are verified via the browser preview** (`netlify dev` + the preview tools / webapp-testing), not unit tests. **Backend pure-logic tasks are TDD with Vitest.**
- **Working name "VoiceLab"** appears only in the shell header, `<title>`, and repo name.

### Task 4: Projects backend (CRUD + token minting)

**Files:**
- Create: `netlify/functions/projects.mjs`

**Interfaces:**
- Consumes: `readJSON/writeJSON`, `PROJECTS_KEY` (Task 3).
- Produces: HTTP endpoints on `/.netlify/functions/projects`:
  - `GET` → `Project[]` (auto-creates a default "My First Project" if none exist).
  - `POST {name}` → new `Project` (mints `webhookToken` = `crypto.randomUUID().replace(/-/g,"")`).
  - `PATCH {id, name}` → updated `Project`.
  - `DELETE ?id=` → `{ok:true}` (also deletes that project's calls/transcripts blobs).

- [ ] **Step 1: Implement `projects.mjs`**

```js
import { readJSON, writeJSON, PROJECTS_KEY, listKeys, getVoiceStore } from "../lib/store.mjs";

const json = (data, status = 200) =>
  new Response(JSON.stringify(data), { status, headers: { "content-type": "application/json" } });

function newProject(name) {
  return {
    id: crypto.randomUUID(),
    name: name?.trim() || "Untitled project",
    webhookToken: crypto.randomUUID().replace(/-/g, ""),
    createdAt: new Date().toISOString(),
  };
}

export default async (req) => {
  const method = req.method;
  let projects = await readJSON(PROJECTS_KEY, []);

  if (method === "GET") {
    if (projects.length === 0) {
      projects = [newProject("My First Project")];
      await writeJSON(PROJECTS_KEY, projects);
    }
    return json(projects);
  }
  if (method === "POST") {
    const { name } = await req.json();
    const p = newProject(name);
    projects.push(p);
    await writeJSON(PROJECTS_KEY, projects);
    return json(p, 201);
  }
  if (method === "PATCH") {
    const { id, name } = await req.json();
    const p = projects.find((x) => x.id === id);
    if (!p) return json({ error: "not found" }, 404);
    p.name = name?.trim() || p.name;
    await writeJSON(PROJECTS_KEY, projects);
    return json(p);
  }
  if (method === "DELETE") {
    const id = new URL(req.url).searchParams.get("id");
    projects = projects.filter((x) => x.id !== id);
    await writeJSON(PROJECTS_KEY, projects);
    const store = getVoiceStore();
    for (const key of await listKeys(`calls/${id}/`)) await store.delete(key);
    for (const key of await listKeys(`transcripts/${id}/`)) await store.delete(key);
    return json({ ok: true });
  }
  return json({ error: "method not allowed" }, 405);
};
```

```js
import { timingSafeEqual } from "node:crypto";
const eq = (a, b) => {
  const ab = Buffer.from(a), bb = Buffer.from(b);
  return ab.length === bb.length && timingSafeEqual(ab, bb);
};
export default async (req) => {
  const expected = process.env.SITE_PASSWORD || "";
  const { password } = await req.json();
  if (!expected || !eq(password || "", expected))
    return new Response(JSON.stringify({ error: "invalid" }), { status: 401 });
  return new Response(JSON.stringify({ ok: true }), {
    status: 200,
    headers: { "set-cookie": `vl_access=${expected.length}-ok; Path=/; HttpOnly; SameSite=Lax; Max-Age=2592000` },
  });
};
```

- [ ] **Step 3: Implement `parseRetellEvent` + `mergeCall`**

```js
const iso = (ms) => (typeof ms === "number" ? new Date(ms).toISOString() : null);

export function parseRetellEvent(body) {
  const eventType = body?.event ?? body?.type ?? "unknown";
  const call = body?.call ?? body?.data ?? {};
  const callId = call.call_id ?? body?.call_id ?? null;
  const analysis = call.call_analysis ?? null;
  const durationSeconds =
    typeof call.duration_ms === "number" ? Math.round(call.duration_ms / 1000)
    : typeof call.start_timestamp === "number" && typeof call.end_timestamp === "number"
      ? Math.round((call.end_timestamp - call.start_timestamp) / 1000) : null;

  const fields = {
    direction: call.direction ?? null,
    fromNumber: call.from_number ?? null,
    toNumber: call.to_number ?? null,
    startedAt: iso(call.start_timestamp),
    endedAt: iso(call.end_timestamp),
    durationSeconds,
    recordingUrl: call.recording_url ?? null,
    transcript: typeof call.transcript === "string" ? call.transcript : null,
    disconnectReason: call.disconnection_reason ?? null,
    summary: typeof analysis?.call_summary === "string" ? analysis.call_summary : null,
    sentiment: typeof analysis?.user_sentiment === "string" ? analysis.user_sentiment : null,
    callSuccessful: typeof analysis?.call_successful === "boolean" ? analysis.call_successful : null,
    callAnalysis: analysis ?? null,
    _outcome: eventType !== "call_started" ? (call.call_status ?? eventType) : null,
    rawPayload: body,
  };
  return { eventType, callId, fields };
}

export function mergeCall(existing, fields, eventType) {
  const base = existing ?? {
    callId: null, projectId: null, direction: null, fromNumber: null, toNumber: null,
    startedAt: null, endedAt: null, durationSeconds: null, recordingUrl: null, transcript: null,
    outcome: null, disconnectReason: null, summary: null, sentiment: null, callSuccessful: null,
    callAnalysis: null, rawPayload: null, receivedAt: new Date().toISOString(), rating: null,
  };
  const out = { ...base };
  for (const [k, v] of Object.entries(fields)) {
    if (k === "_outcome") { if (v !== null && v !== undefined) out.outcome = v; continue; }
    if (k === "rawPayload") { out.rawPayload = v; continue; }
    if (v !== null && v !== undefined) out[k] = v;
  }
  return out;
}
```

- Consumes: `parseRetellEvent`, `mergeCall`, `verifyHmac` (Tasks 8–9); `readJSON/writeJSON`, `callKey`, `PROJECTS_KEY` (Task 3).
- Endpoint: `POST /.netlify/functions/retell-webhook?token=<projectWebhookToken>`.

```js
import { parseRetellEvent, mergeCall, verifyHmac } from "../lib/retell.mjs";
import { readJSON, writeJSON, callKey, PROJECTS_KEY } from "../lib/store.mjs";

export default async (req) => {
  if (req.method !== "POST") return new Response("method not allowed", { status: 405 });
  const token = new URL(req.url).searchParams.get("token");
  const projects = await readJSON(PROJECTS_KEY, []);
  const project = projects.find((p) => p.webhookToken === token);
  if (!project) return new Response("unknown token", { status: 401 });

  const raw = await req.text();
  const sig = req.headers.get("x-retell-signature");
  const v = await verifyHmac(raw, sig, process.env.RETELL_API_KEY);
  if (!v.ok) return new Response(`signature: ${v.reason}`, { status: 401 });

  let body;
  try { body = JSON.parse(raw); } catch { return new Response("bad json", { status: 400 }); }

  const { eventType, callId, fields } = parseRetellEvent(body);
  if (!callId) return new Response("no call_id", { status: 200 }); // ack, nothing to store

  const key = callKey(project.id, callId);
  const existing = await readJSON(key, null);
  const merged = mergeCall(existing, fields, eventType);
  merged.callId = callId;
  merged.projectId = project.id;
  await writeJSON(key, merged);
  return new Response(JSON.stringify({ ok: true }), { status: 200, headers: { "content-type": "application/json" } });
};
```
